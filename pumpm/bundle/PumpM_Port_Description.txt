2. Pump Adapter (PumpAController)
	input
 		receiving_pumpa		PumpPreamble	WorkingMessage	
	 		WorkingMessage ID List
	 			ACK
	 				- 일반 환경정보 설정 응답 (P2)
	 				- 주유기 환경 정보 설정 응답 (P3)
	 				- ODT 환경 정보 설정 (셀프 , 충전기) (P5)
	 				- 영업일 및 시간 설정 (P6)
	 				- 충전기 ODT POS 환경정보 설정 (PT)
	 				
	 				- 노즐 제어 명렁 응답 (주유기,셀프,충전기) (PA)
	 				- 정액/정량 설정 응답 (주유기,셀프,충전기) (PB)
	 			S3	주유/충전 중 자료 전송 (주유기,셀프,충전기)
	 			S4	주유완료 자료 전송 (주유기, 셀프)
	 				: Issue -> 충전기 주유완료 정보는 ?
	 			HF	Preset 자료 전송
	 				: Issue -> 주유기 Adapter 로 부터 올 필요 없음. 
	 					( 이 정보는 이미 Preset 정보를 관리하는 테이블로 부터 정보를 받아서 보냄)
	 			HB	소모셀프 외상거래 승인 요청
	 			HA	소모셀프 카드결재 승인 요청
	 			HC	소모셀프 주유 완료시 결재 정보 전송
	 				다쓰노 셀프 주유 완료시 결재 정보 전송
	 			HE	다쓰노 셀프 카드 결재 승인 요청
	 			S9	고객 카드 승인 요청
	 			SB	충전기 카드 결재 승인 요청
	 			SH	충전기 ODT 최종 판매 데이터 전송
	 				: Issue -> SH 전문을 사용하여 주유완료 (S4) 전문을 만들어서 Adapter 가 Module 에게 보내어야 함.
	 			BA	보너스 점수 누적 요청
	 			TJ	현금 영수증 요청
	 			SF	보관증 번호 부여 요청
	 			SG	보관량 조회 요청
	 			S5	토털게이지 전송 (Adapter 가 Module 에게 보내지만, 이 내용은 Skip)
	 
	 			SE	주유 디바이스 이상 정보 전송
	 				receiving_pumpa Data 를 그대로 StateMController 로 보낸다.
	 			S8	주유기/충전기 상태 전송
	 				receiving_pumpa Data 를 그대로 StateMController 로 보낸다. 			
	 				
	output
